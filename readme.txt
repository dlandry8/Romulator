To use testPinsArduino.ino:

1.) Download to a folder named testPinsArduino.
2.) Upload to an Arduino Mega 2560 Rev 3.
3.) Open up the Serial Monitor in the Arduino IDE with the Arduino Mega 2560 attached. 
	Make sure the serial monitor is monitoring the correct com.
4.) From the monitor, you may enter a pin number to test that pin's connection on the board.


To use testSerialArduino.ino:

1.) Download to a folder named testSerialArduino.
2.) UPload to an Arduino Mega 2560.
3.) Use either an external serial program or the Arduino Monitor to see if the Arduino will echo your input.
